## Como Rodar?
### Instale as dependências
```
npm i
```
### Instale o Angular CLI
```
npm i -g @angular/cli
```
### Rode no LocalHost
```
ng serve --open
```
### Construa a Aplicação para uso estático
```
ng build
```
